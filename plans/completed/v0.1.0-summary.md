# v0.1.0 - Hardware Info Detection (COMPLETED)

**Status:** ✅ Completed (2025-12-25)
**Tech Stack:** Rust 1.92 + sysinfo 0.37 + clap 4.5

## Summary

Cross-platform hardware detection CLI tool that displays CPU, GPU, RAM, and Disk information.

## Features Implemented

| Component | Detection Method | Status |
|-----------|------------------|--------|
| Platform | `std::env::consts::OS` + system commands | ✅ |
| CPU | `sysinfo::System` (model, cores) | ✅ |
| GPU | macOS: `system_profiler`<br>Windows: PowerShell<br>Linux: `lspci` | ✅ |
| RAM | `sysinfo::System` (total, used) | ✅ |
| Disk | `sysinfo::System` (name, size) | ✅ |
| Multi-language | Vietnamese/English | ✅ |

## Files Created

- `src/main.rs` - CLI entry point with clap
- `src/platform/mod.rs` - Platform detection trait
- `src/hw/` - Hardware detection modules (cpu, gpu, ram, disk)
- `src/fmt.rs` - Output formatting
- `src/lang.rs` - Multi-language support
- `src/prompt.rs` - Language selection prompt

## Binary Size

~550KB (optimized release build)
